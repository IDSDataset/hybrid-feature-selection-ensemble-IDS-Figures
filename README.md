# IEEE TDSC Paper: "Evaluation to Integration: Hybrid Feature Selection Framework with Ensemble Machine Learning for Intrusion Detection"

This repository contains the **PDF document** for the paper titled *"Evaluation to Integration: Hybrid Feature Selection Framework with Ensemble Machine Learning for Intrusion Detection"*, published in **IEEE Transactions on Dependable and Secure Computing (TDSC)**.

The paper presents a framework for integrating **hybrid feature selection** with **ensemble machine learning models** to improve the performance of **Intrusion Detection Systems (IDS)**. The approach combines multiple feature selection techniques to enhance the detection of network intrusions.

## Key Highlights:
- **Hybrid Feature Selection**: A framework that combines the strengths of different feature selection techniques to improve the robustness of intrusion detection.
- **Ensemble Machine Learning**: The integration of ensemble learning models to improve the accuracy and reliability of IDS.
- **Performance Evaluation**: Comparative analysis using multiple evaluation metrics like **ROC curves**, **confusion matrices**, **performance metrics**, and **learning curves**.
- **Experimental Setup**: Comprehensive experiments on the CIC-IDS-2017 dataset, measuring prediction time, accuracy, and scalability.

## Contents:

- **figures_collection.pdf**: The **compiled PDF document** that includes all the figures, charts, and descriptions presented in the paper. This document illustrates various performance comparisons, hybrid feature selection methodologies, model evaluation metrics, and experimental results.

## Figures Included:
The PDF document features various visualizations that help in understanding the proposed methodology:
- **Interaction maps** between IDS and networks, including machine learning integration.
- **Feature selection techniques** for ensemble machine learning models.
- **Performance metrics** and comparisons of ensemble models.
- **ROC curves** and **confusion matrices** for evaluating model performance.
- **Learning curves** and **prediction time measurements** for evaluating model efficiency.
- **Scaling characteristics** of the IDS pipeline.
